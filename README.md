# अक्षर  Akshar — Traditional Script Recognition

A beautifully crafted Flutter app for digit and handwritten text recognition,
designed with a traditional Indian aesthetic.

**Color Palette** (colorhunt.co/palette/5dd3b66e5034cdb885efe1b5)

- 🟡 `#CDB885` — Antique Gold
- 🟡 `#EFE1B5` — Parchment Cream

## 🚀 Setup

### 1. Get dependencies
```bash
flutter pub get
```

### 2. Download fonts
Download **Tiro Devanagari Hindi** from Google Fonts:
👉 https://fonts.google.com/specimen/Tiro+Devanagari+Hindi

Place the files in `assets/fonts/`:
```
assets/fonts/
├── TiroDevanagariHindi-Regular.ttf
└── TiroDevanagariHindi-Italic.ttf
```

### 3. Run
```bash
# iOS (open simulator first)
open -a Simulator
flutter run

# Android
flutter run
```

For iOS, run `cd ios && pod install && cd ..` first.

## ✨ Features

| Feature | Description |
|---|---|
| ✍️ **लिखना  Draw** | Freehand canvas, gold strokes, traditional corner ornaments |
| 📷 **कैमरा  Camera** | Capture and OCR recognize instantly |

| 🔢 **अंक  Digit** | Extract digit sequences |

## 🎨 Design Language

- **Deep mahogany** background with warm wood tones
- **Antique gold** strokes on the drawing canvas
- **Jade teal** for interactive elements and live indicators
- **Parchment cream** for primary text
- **Devanagari script** labels alongside English
- Traditional corner ornaments on the canvas
- Rangoli-inspired dot grid background pattern
